HTML Basic Elements

Etiquetas Meta :

meta http-equiv="X-UA-Compatible" content="IE=edge"

http-equiv Define reglas para la pw.
Content-Security-Policy (Define un enlace a las políticas de contenido de una página web).
refresh (Permite actualizar la página web cada N segundos, o incluso redirigir a otra URL).
Ejemplo 1 meta http-equiv="refresh" content="60"
Ejemplo 2 meta http-equiv="refresh" content="0; url=https://cssreference.io"
X-UA-Compatible (Define en qué versión de Internet Explorer se debe representar la página web.
Notas: content Define el contenido de los metadatos. Esto varía según el valor name o http-equiv.

meta name="viewport" content="width=device-width, initial-scale=1.0"

name Define información adicional adjunta a la página web.
viewport viewport (Define reglas de dimensión y escala para la ventana gráfica).

Link (css)

link rel="stylesheet" href="" type=""/

Generalmente es para vincular hojas de estilo. y se ubica en la etiqueta head

  1. rel
    2. Define el tipo de enlace:
    • "stylesheet" Hoja de estilos CSS
    • "author" Datos del Autor
    • "icon" Favicon
    • "next" Pag. siguiente
  2. href
    3. Define una URL para el enlace
    • "https://www.google.com" URL Absoluta
    • ../main/index.html URL Relativa
  3. type
    4. Tipo de recurso vinculado:
    • text/css
    • text/html

Enlaces

HTML links are defined with the a tag:

[a href="./ejemplo.html" target="_blank"] This is a link [/a]

Si target="_blank", el enlace se abre en una nueva ventana de window o en otra pestaña. Podemos usar cualquier elemento en un enlace una imagen un texto un icono etc.

Elementos Básicos de un Formulario

form
---Engloba el formulario
label
---Nombre del campo a rellenar, nota para unir un label a un input debemos ponerle al label el atributo for="label name" y en el atributo input poner el atributo id="label name"
input
---Campo que le permite al usuario interactuar
button
---Crea un boton para enviar el formulario

Formlario

HTML Form Attributes

action
define la acción que se realizará cuando se envíe el formulario.
target
especifica dónde mostrar la respuesta que se recibe después de enviar el formulario.

Posibles valores del atributo TARGET

  1. _blank
    La respuesta se muestra en una nueva ventana o pestaña
  2. _self
    La respuesta se muestra en la ventana actual
  3. _parent
    La respuesta se muestra en el marco principal.
  4. _top
    La respuesta se muestra en todo el cuerpo de la ventana.
  5. framename
    La respuesta se muestra en un iframe con nombre
